# PERN-Projects

A collection of full-stack web applications built with the **PERN** stack:

- **P**ostgreSQL — relational database
- **E**xpress — Node.js web framework
- **R**eact — frontend UI library
- **N**ode.js — JavaScript runtime

## Projects

| # | Project | Description |
|---|---------|-------------|
| 1 | [pern-todo-app](./pern-todo-app) | Full-stack Todo CRUD application |

## Tech Stack

- **Frontend:** React (Vite), Bootstrap
- **Backend:** Node.js, Express
- **Database:** PostgreSQL
- **Other:** dotenv, cors, pg (node-postgres)

## Getting Started

Each project has its own `README.md` with setup instructions. Navigate into the project folder and follow the steps there.

```bash
cd pern-todo-app
```
